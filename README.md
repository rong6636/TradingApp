一個基於資料截取、網路架構、任務排程，架設於雲端伺服器的DISCORD生活訊息機器人

## 主要功能

- 每三小時提供新聞關鍵字
- 第一時間提供氣象局地震報告資訊
- 每小時更新天氣警特報資訊
- 與朋友的玩物 - 指令抽圖 :)
- 普通功能：CPC油價、小遊戲、好友呼叫...等
- 有些過時，如：奧林匹克排行...等

隨需求增加，機器人會不斷增加功能。
